# QuantumDGM Artifact Layout

This repository separates reproducibility artifacts into two locations:

- `experiments/results/`: raw machine-readable experiment outputs (`*.json`).
- `figures/`: paper-facing assets (figures and generated LaTeX table macros/metadata).

## Paper-facing files

- Figures (PNG): generated by experiment plot routines in `experiments/`.
- `paper_generated_rows.tex`: generated table rows/macros for `paper.tex`.
- `paper_generated_metadata.tex`: generation timestamp macro for `paper.tex`.
- `paper_table_metrics.json`: aggregated table metrics used for paper reporting.

## Commands

Run experiments:

```bash
python -m experiments.run_all_for_paper --fast
```

Regenerate paper assets:

```bash
python -m experiments.generate_paper_artifacts
```

After generation, compile `paper.tex` with assets from `figures/`.
